# 校园卡守护者

隐私保护型校园卡智能匹配与归还微信小程序。当前版本可用微信开发者工具以游客模式打开，页面使用本地演示数据；取得 AppID 后可接入微信云开发。

## 本地运行

1. 执行 `npm install`。
2. 打开微信开发者工具，导入本目录。
3. 暂无 AppID 时使用测试号；取得 AppID 后修改 `project.config.json`。
4. 执行 `npm run verify` 检查测试、类型、代码风格和格式。

## 云端开通（需要项目管理员本人操作）

1. 在微信公众平台注册小程序并取得 AppID。
2. 在开发者工具中开通云开发，分别创建开发与生产环境。
3. 开通腾讯云 OCR，创建仅允许调用 OCR 的最小权限子账号。
4. 在云函数环境变量中配置 `STUDENT_HMAC_SECRET`、`TENCENT_SECRET_ID`、`TENCENT_SECRET_KEY`；禁止写入源码。
5. 按 `docs/cloud-setup.md` 创建集合、索引和权限。

## 隐私原则

- 原图只用于 OCR 与脱敏，成功后立即删除。
- 学号使用服务端 HMAC 匹配，不作为公开字段保存。
- 公开列表只展示脱敏姓名、脱敏学号与脱敏图片。
- AI 审核能力默认关闭，只有管理员配置预算后才能启用。
