# 校园卡智能招领小程序设计

本项目面向单校多校区，以隐私优先方式完成拾卡发布、失卡登记、自动匹配、安全认领和归还。技术栈为原生微信小程序、TypeScript 与微信云开发。

首版默认采用规则审核：学号 HMAC 精确比对，姓名、校区、地点、时间和非公开卡片特征参与评分。高置信申请自动通过，明确冲突自动拒绝，灰区、多人竞争和申诉才进入管理员复核。腾讯混元适配器只预留，默认关闭。

原图仅用于 OCR 和脱敏，处理完成后删除；公开数据只包含脱敏姓名、脱敏学号和脱敏缩略图。OCR 失败或额度耗尽时允许人工填写。

首版页面包含首页、拾卡发布、失卡查询/登记、认领、交接、消息、个人中心和小程序内管理页。不包含地图、自由聊天、排行榜、智能柜和独立网页后台。
